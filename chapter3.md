# HTML   
* html은 마크업 언어라고 부른다  
* 꺽쇠<>안에 넣어서 코드를 작성  
* 태그 문법  

```html
<div 속성이름="속성값"></div>  
<div style="color:red"></div>  
```  

# 자주 사용되는 html 태그  
```html
    <h1>헤드 태그</h1>
    <h2>1에서 6까지</h2>
    <h6>점점 작아짐</h6>
    <div>한 줄 씩 영역을 가짐</div>
    <div>영역2</div>
    <p>긴 문장을 쓸 때</p>
    <p>긴 <span style="color:red">문장 중간 특정 단어를 꾸미고 싶을 때</span> 문장을 쓸 때</p>
    <ul>
        <li>리스트1</li>
        <li>리스트2</li>
    </ul>
    <br/> //영역 사이 공백
    <hr/> //영역 사이 수평선과 공백
    <input type="text" /> //입력 받을 때
    <input type="number" />
    <button>버튼</button>
    <img src="이미지 주소"/>
```
